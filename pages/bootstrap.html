Guía para implementar Bootstrap a tu proyecto

¿Qué es Bootstrap?

Bootstrap es un framework de CSS de código abierto que facilita el diseño de sitios web y aplicaciones web responsive. Nos proporciona una gran cantidad de clases listas para usar que permiten maquetar interfaces limpias y modernas de forma rápida y sencilla. Ofrece una colección de componentes y estilos predefinidos, como sistemas de cuadrícula, botones, formularios y barras de navegación, que facilitan la construcción de interfaces de usuario adaptables a diferentes dispositivos.

En esta guía aprenderás a integrar Bootstrap en un proyecto Java utilizando Maven y JSP.

1. Crear un proyecto Maven con soporte web

Puedes crear el proyecto desde tu IDE favorito (Eclipse, IntelliJ, NetBeans) o desde la terminal usando Maven Archetype.

Asegúrate de que tenga la siguiente estructura: 

src/
├── main/
│   ├── java/
│   ├── resources/
│   └── webapp/
│       ├── WEB-INF/
│       └── index.jsp

2. Integrar Bootstrap vía CDN

Esta es la forma más rápida de usar Bootstrap. Simplemente agrega el siguiente enlace en la etiqueta <head> de tu archivo .jsp: 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

También puedes agregar el JS opcional de Bootstrap al final del <body> si deseas usar componentes interactivos:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

3. Integrar Bootstrap usando WebJars

También puedes usar WebJars para gestionar las dependencias desde Maven.

Agrega esta dependencia en tu pom.xml:

<dependency>
    <groupId>org.webjars</groupId>
    <artifactId>bootstrap</artifactId>
    <version>5.3.3</version>
</dependency>

Y luego enlaza Bootstrap desde tu JSP:

<link rel="stylesheet" href="${pageContext.request.contextPath}/webjars/bootstrap/5.3.3/css/bootstrap.min.css">

4. Crear un archivo JSP con Bootstrap

Aquí tienes un ejemplo básico de cómo quedaría tu index.jsp:

<%@ page contentType="text/html;charset=UTF-8" %>
<!DOCTYPE html>
<html>
<head>
    <title>Mi Proyecto con Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body class="container mt-5">
    <h1 class="text-primary">¡Hola desde JSP con Bootstrap!</h1>
</body>
</html>
