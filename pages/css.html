Guía para implementar CSS a tu proyecto

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que describe cómo se presentan los elementos de un documento HTML. Nos permite personalizar el aspecto visual de una página web, controlando colores, fuentes, márgenes, posiciones y diseños de forma flexible.

En esta guía aprenderás a integrar tu propio archivo CSS en un proyecto Java utilizando Maven y JSP.

1. Crear un proyecto Maven con soporte web

Asegúrate de que tenga la siguiente estructura: 


src/
├── main/
│   ├── java/
│   ├── resources/
│   └── webapp/
│       ├── WEB-INF/
│       ├── styles/
│       │   └── style.css
│       └── index.jsp

2. Crear un archivo CSS

Dentro de la carpeta styles o el nombre que quieras pero que sepas que ese es el estilo de preferencia style, crea un archivo llamado style.css o (el nombre que quieras) con tus reglas de estilo Ejemplo. 

body {
    font-family: Arial, sans-serif;
    background-color: #f8f9fa;
    margin: 0;
    padding: 0;
}

h1 {
    color: #007bff;
    text-align: center;
}

3. Enlazar el CSS desde tu JSP

En tu archivo JSP, dentro de la etiqueta <head>, agrega: 

<link rel="stylesheet" href="${pageContext.request.contextPath}/styles/style.css">

4. Crear un archivo JSP con CSS

Ejemplo básico de index.jsp enlazando tu CSS:

<%@ page contentType="text/html;charset=UTF-8" %>
<!DOCTYPE html>
<html>
<head>
    <title>Mi Proyecto con CSS</title>
    <link rel="stylesheet" href="${pageContext.request.contextPath}/styles/style.css">
</head>
<body>
    <h1>¡Hola desde JSP con CSS puro!</h1>
</body>
</html>
