EJB - Enterprise JavaBeans

EJB (Enterprise JavaBeans) es una tecnología de Java utilizada para desarrollar componentes de software del lado del servidor que encapsulan la lógica de negocio de una aplicación empresarial. Los EJB se ejecutan dentro de un contenedor especial que ofrece servicios como seguridad, transacciones, persistencia y manejo de concurrencia, permitiendo que el desarrollador se concentre en la lógica de la aplicación sin preocuparse por estos aspectos técnicos. Son ideales para aplicaciones grandes y distribuidas, ya que facilitan la creación de sistemas escalables, seguros y con un manejo eficiente de los recursos del servidor.

EJB (Enterprise JavaBeans) es una tecnología de Java EE que permite desarrollar componentes distribuidos, escalables, seguros y transaccionales en aplicaciones empresariales.

Su principal ventaja es facilitar el desarrollo de la lógica del negocio separándola de la lógica de presentación o de acceso a datos.

Para crear y trabajar con EJB en NetBeans 21, puedes seguir estos pasos:

1. Abre NetBeans y selecciona "Nuevo Proyecto".
2. Elige la categoría Java con Maven y selecciona Enterprise Application, luego haz clic en Next.
3. Define los valores de Group ID y Artifact ID, y elige un nombre adecuado para tu proyecto.
4. Asegúrate de que esté habilitado el módulo EJB Module en la configuración del proyecto.
5. Una vez creado el proyecto, haz clic derecho en el módulo EJB y selecciona "Nuevo > Sesión Bean" para crear un EJB de sesión.
6. Define el tipo de bean: Stateless (sin estado), Stateful (con estado) o Singleton (instancia única).
7. Escribe la lógica del negocio dentro de la clase EJB. Puedes usar anotaciones como @Stateless o @Stateful.
8. Agrega interfaces remotas o locales si deseas exponer los métodos del EJB a otros módulos o aplicaciones.
9. Usa un cliente (como un servlet o aplicación web) para inyectar el EJB y llamar a sus métodos.
10. Compila y despliega tu aplicación en un servidor compatible con Java EE como GlassFish o WildFly.
