Guía para implementar HTML en tu proyecto

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y presentar contenido en la web. Nos permite organizar textos, imágenes, enlaces, tablas, formularios y otros elementos para que los navegadores puedan mostrarlos correctamente.

En esta guía crearas y aprenderás los conceptos mas basicos y comunes ademas cómo organizar tu código HTML en un proyecto.

1. Estructura básica de un documento HTML

En este ejercicio y en los siguientes tendras que crear archivos .html para poder visualizarlos

Un archivo HTML típico tiene la siguiente estructura: 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es mi primer documento HTML.</p>
</body>
</html>

2. Etiquetas HTML más comunes

Algunas de las etiquetas más utilizadas en HTML son:

  • <h1>...</h1> a <h6>...</h6> - Encabezados
  • <p>...</p> - Párrafos
  • <a href="...">...</a> - Enlaces
  • <img src="..." alt="..."> - Imágenes
  • <ul>, <ol> y <li> - Listas
<h2>Lista de compras</h2>
<ul>
    <li>Pan</li>
    <li>Leche</li>
    <li>Huevos</li>
</ul>

3. Insertar imágenes

Para mostrar imágenes en HTML, usamos la etiqueta <img>: 

<img src="imagenes/foto.jpg" alt="Descripción de la imagen">

El atributo alt es importante para la accesibilidad y en caso de que la imagen no cargue.

4. Tablas y formularios

HTML permite crear tablas y formularios para mostrar datos o recoger información: 

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
    </tr>
</table>
                
<form action="procesar.php" method="post">
    <label>Nombre:</label>
    <input type="text" name="nombre">
    <input type="submit" value="Enviar">
</form>
